# Audio Widget

Windows向けのシンプルで軽量なフローティング音楽コントローラーです。
YouTube (ブラウザ)、Spotify、Windows Media Playerなどの再生状態を検知し、最前面で操作できます。

作業中にウィンドウを切り替えることなく、手元でサッと曲送りやシーク操作を行いたい人のために開発しました。

## 主な機能

  * **ユニバーサルコントロール:** Windowsのシステムメディア連携 (SMTC) を利用し、あらゆるメディアソースを操作可能。
  * **シークバー操作:** 再生位置の表示と、ドラッグによる直感的なシーク。
  * **3つのサイズ:** 邪魔にならない `Small`、標準的な `Medium`、アルバムアート重視の `Large`。
  * **カスタマイズ:**
      * ライト / ダーク / システム連動テーマ
      * ウィンドウの透明度調整
      * 最前面表示 (Always on Top) の切り替え
  * **完全なウィンドウ制御:** ウィンドウ位置の固定 (Lock) や、マウスホバー時の挙動制御。

## 動作環境

  * OS: Windows 10 / 11
  * 必須ランタイム: .NET 8 Desktop Runtime
      * 起動しない場合は [こちら](https://dotnet.microsoft.com/en-us/download/dotnet/8.0) からインストールしてください。

## インストールと起動方法

1.  最新の `AudioWidget.zip` をダウンロードします。
2.  zipファイルを任意のフォルダに\*\*展開（解凍）\*\*してください。
3.  フォルダ内の `AudioWidget.exe` をダブルクリックして起動します。

### 「WindowsによってPCが保護されました」と表示される場合

本ソフトウェアは、初回起動時にWindowsのSmartScreen機能により警告が表示される場合があります。

ウイルスではありませんので、以下の手順で起動してください：

1.  警告画面にある **「詳細情報」** というリンクをクリックします。
2.  画面右下に現れる **「実行」** ボタンをクリックします。

※ 一度実行すると、次回からは表示されなくなります。

## 操作方法

  * **再生/一時停止、曲送り/戻し:** 各アイコンをクリック。
  * **移動:** ウィンドウの背景部分をドラッグ。
  * **設定メニュー:** ウィンドウを **右クリック** するとメニューが開きます。
      * レイアウト変更 (小/中/大)
      * テーマ変更
      * 透明度設定
      * 移動のロック / 最前面固定
  * **終了:** 右クリックメニューの「閉じる」を選択。
